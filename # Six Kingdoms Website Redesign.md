# Six Kingdoms Website Redesign

A professional, modern redesign of the Six Kingdoms landscaping and permaculture website.

## Features

- ✨ Modern, clean design with nature-themed colors
- 📱 Fully responsive (works on all devices)
- 🎨 Beautiful typography using Montserrat and Lora fonts
- 🌿 Project-focused layout showcasing past work
- 💚 Smooth animations and transitions
- ♿ Accessible and SEO-friendly

## Pages Included

1. **index.html** - Homepage with hero, services, featured projects, and contact form
2. **projects.html** - Complete archive of past projects
3. **land-management.html** - Holistic land management services
4. **ecopools.html** - Natural swimming pools information

## How to Use

1. Open `index.html` in any modern web browser
2. Navigate between pages using the top navigation menu
3. All content from the original site is preserved

## Adding Images

To add your own images:
1. Create an `images` folder in this directory
2. Add your project photos
3. Update the image paths in the HTML files
